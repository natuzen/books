# Época & Rosa

**Tu guía de romance literario histórico y clásico.**

Época & Rosa es un sitio web completo dedicado a las amantes de las novelas románticas históricas y clásicas, construido con Astro 5, Preact, Tailwind CSS y TypeScript.

## Características

- 📚 **Catálogo de libros** — 20 novelas románticas clásicas con fichas detalladas
- 🔍 **Motor de búsqueda** — Filtra por autor, categoría, tropo e intensidad romántica
- 🎯 **Quiz literario** — Descubre qué tipo de lectora romántica eres
- 💌 **Generador de citas** — Citas inmortales de Jane Austen, Emily Brontë y más
- 🗺️ **Mapa literario interactivo** — Explora escenarios reales de las novelas (Leaflet)
- 🌸 **Recomendador por estado de ánimo** — Libros según cómo te sientes
- 🏆 **Rankings con votaciones** — Vota por tus novelas favoritas
- 📅 **Calendario literario** — Efemérides literarias románticas
- 🎬 **Tendencias** — Próximos estrenos y novedades editoriales
- 📋 **Listas temáticas** — Colecciones curadas con contenido editorial

## Stack tecnológico

- **[Astro 5](https://astro.build/)** — Framework web estático
- **[Preact](https://preactjs.com/)** — Componentes interactivos
- **[Tailwind CSS](https://tailwindcss.com/)** — Estilos utilitarios
- **[Leaflet](https://leafletjs.com/)** — Mapas interactivos
- **TypeScript** — Tipado estático

## Desarrollo local

```bash
npm install
npm run dev
```

## Construcción

```bash
npm run build
npm run preview
```

## Estructura del proyecto

```
/
├── data/                    # Datos JSON (libros, citas, etc.)
├── public/                  # Archivos estáticos
└── src/
    ├── components/          # Componentes Astro y Preact (.tsx)
    ├── content/listas/      # Contenido editorial en Markdown
    ├── layouts/             # Layouts de página
    ├── pages/               # Rutas del sitio
    ├── styles/              # CSS global
    └── utils/               # Utilidades TypeScript
```
